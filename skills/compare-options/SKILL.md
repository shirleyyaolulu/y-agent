# Compare Options

When comparing options:

1. Name the options being compared.
2. Explain what each option means.
3. Compare them using 2-4 practical criteria.
4. State which option is better for the current project stage.
5. Explain the trade-off behind the recommendation.
6. Keep the conclusion concrete and actionable.
