# Explain Concept

When explaining a concept:

1. Start with a short plain-language definition.
2. Explain why the concept exists.
3. Show how it fits into the current project or topic.
4. Use a small concrete example.
5. Mention one common misunderstanding.
6. Keep the explanation focused on learning, not implementation detail.
